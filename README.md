# Calculadora de Cálculo II

## Descripción

Proyecto desarrollado en Python que permite realizar operaciones matemáticas relacionadas con Cálculo II mediante una interfaz gráfica.

La aplicación permite calcular derivadas, integrales, límites, evaluar funciones y representar gráficamente funciones matemáticas.

## Tecnologías utilizadas

- Python
- Tkinter
- SymPy
- NumPy
- Matplotlib

## Funciones principales

### Derivadas
Calcula la derivada de una función respecto a la variable x.

Ejemplo:

f(x)=x^3+2x

Resultado:

3x²+2


### Integrales indefinidas

Calcula la integral de una función.

Ejemplo:

∫x² dx

Resultado:

x³/3 + C


### Integrales definidas

Permite ingresar límites inferior y superior.

Ejemplo:

∫(a,b) f(x) dx


### Límites

Calcula el límite de una función cuando x se acerca a un punto.


### Evaluación de funciones

Permite reemplazar un valor en una función.

Ejemplo:

f(x)=x²

x=5

Resultado:

25


### Gráficas

Representa visualmente la función ingresada.

## Instalación

Instalar dependencias:

pip install sympy numpy matplotlib


Ejecutar:

python calculadora.py
